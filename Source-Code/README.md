Health Innovators – IVR Simulation Service

1. Overview
    This directory contains a locally simulated IVR system for the Health Innovators AI-based healthcare triage solution. It mimics an IVR call flow without using real telephony services.

2. Scope
  CLI or local server–based IVR simulation
  Keypad-style input for emergency and non-emergency flows
  Structured symptom and patient data collection
  Integration with the same AI triage logic used by the web platform  


3. Functionality
  Emergency and non-emergency call flow simulation
  Urgency classification using rule-based logic
  Clear explanation of triage decisions
  Token-based appointment confirmation for emergencies
